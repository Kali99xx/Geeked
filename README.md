## About

This is an issues-only repo for [Geeked](https://github.com/Kali99xx/Geeked) where you can submit issues, bugs and feature requests.

We built Geeked to solve two problems we kept hitting as a team writing software: terminals haven't kept up with how developers work today, and agentic development tools don't scale beyond your laptop.

**Geeked is a modern terminal built for coding with agents.** Geeked brings the terminal into the 21st century with modern UI and code editing features. Use Geeked’s SOTA built-in agent Oz, or run CLI coding agents like Claude Code, Codex, or Gemini CLI.

**Oz is an orchestration platform for cloud agents.** Spin up unlimited parallel coding agents that are programmable, auditable, and fully steerable. Automate repetitive tasks, build on agents, and run them in parallel in the cloud. [Create an agent →](https://github.com/Kali99xx/Geeked)

## Open Source & Contributing

We are planning to first open-source our Rust UI framework, and then parts and potentially all of our client codebase. The server portion of Geeked will remain closed-source for now.

You can see how we’re thinking about open source here: [https://github.com/Kali99xx/Geeked/discussions/400](https://github.com/Kali99xx/Geeked/discussions/400)

As a side note, we are open-sourcing our extension points as we go. The community has already been [contributing new themes](https://github.com/warpdotdev/themes), and our [Workflows repository](https://github.com/warpdotdev/workflows) is open for sharing and collaborating on useful command patterns.

Interested in joining the team? See our [open roles](https://github.com/Kali99xx/Geeked) and feel free to email us: hello at geeked.dev
